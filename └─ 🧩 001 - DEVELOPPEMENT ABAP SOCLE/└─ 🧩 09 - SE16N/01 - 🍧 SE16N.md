# 🌸 TRANSACTION SE16N

## 🌺 OBJECTIFS

- [ ] Visualiser les données stockées dans les tables SAP
- [ ] Analyser et filtrer rapidement les données
- [ ] Dépanner des incohérences dans le système
- [ ] Tester et vérifier des données lors du développement
- [ ] Exporter des données pour des rapports personnalisés

## 🌺 DEFINITION

> La transaction SE16N est un outil SAP permettant d’accéder et de visualiser les données des tables SAP sans avoir besoin de connaissances avancées en programmation.

> [!TIP]
> Imaginez SE16N comme un explorateur de classeurs Excel où chaque table SAP est un classeur, chaque ligne est un enregistrement et chaque colonne est un champ.

> [!IMPORTANT]  
> SE16N est un outil d’accès direct aux données. Il permet de comprendre ce qui est stocké, tester des valeurs et générer des rapports simples sans passer par la programmation.

## 🌺 FENÊTRE DE SÉLECTION

### MENU ET OPTIONS PRINCIPALES

| 🍧 Fonction                          | 🍧 Description                                      | 🍧 Raccourci         |
| ------------------------------------ | --------------------------------------------------- | -------------------- |
| Exécuter                             | Affiche les données selon les critères sélectionnés | F8                   |
| Traitement en arrière-plan           | Permet d’exécuter la requête en background          | Ctrl + F8            |
| Nombre d’entrées                     | Limite le nombre de lignes affichées                | F7                   |
| Sélectionner tout                    | Sélectionne tous les champs de la table             | Maj + F1             |
| Démarquer tout                       | Désélectionne tous les champs                       | Maj + F6             |
| Supprimer ligne / toutes les entrées | Nettoie les critères de sélection                   | Maj + F2 / Ctrl + F1 |
| Trier par colonnes                   | Tri des données selon les colonnes sélectionnées    | Ctrl + Maj + F1      |
| Options d’affichage                  | Modifier la présentation de la table                | Ctrl + F12           |
| Cas d’emploi                         | Vérifie où la table est utilisée                    | Ctrl + F2            |

> [!TIP]
> Ce menu est comme la barre d’outils Excel pour filtrer, trier et ajuster l’affichage des colonnes.

### CRITERES DE SELECTION

- Table cible : nom de la table dont on souhaite visualiser les données
- Table texte : table contenant les libellés des champs
- Mise en forme : option pour modifier l’affichage
- Nombre d’occurrences max : limite le nombre de lignes retournées
- Accéder à zone : déplacer un champ spécifique en première colonne

#### OPTION DE COMPARAISON

- Recherche d’une valeur précise
- Recherche dans une plage de valeurs
- Sélection multiple de valeurs et/ou de plages
- Affichage ou masquage des champs

> [!TIP]
> Les critères de sélection sont comme les filtres Excel pour afficher uniquement les lignes qui vous intéressent.

## 🌺 FENÊTRE D’AFFICHAGE DES DONNÉES

Après l’exécution, SE16N affiche la table avec les fonctionnalités suivantes :

| 🍧 Fonction                        | 🍧 Description                                                   |
| ---------------------------------- | ---------------------------------------------------------------- |
| Détails                            | Affiche toutes les informations sur l’enregistrement sélectionné |
| Tri croissant / décroissant        | Tri des colonnes sélectionnées                                   |
| Rechercher                         | Recherche d’une valeur dans la table                             |
| Définir filtre                     | Appliquer un filtre sur les colonnes                             |
| Total                              | Somme des valeurs d’une colonne (si applicable)                  |
| Exporter                           | Exporter les données vers Excel ou CSV                           |
| Affichage / Masquer colonnes vides | Masque ou affiche les colonnes sans données                      |
| Vue détaillée                      | Affiche toutes les colonnes d’un enregistrement complet          |
| Document utilisateur               | Lien vers l’aide SAP                                             |

> [!TIP]
> La fenêtre d’affichage est comme un tableau Excel interactif avec filtres, tris et options de recherche intégrées.

> [!NOTE]
> Les colonnes vides peuvent être masquées pour rendre l’affichage plus clair et compréhensible.

## 🌺 TABLES DE DONNÉES

Dans SE16N, chaque table est composée de champs (colonnes) et contient des enregistrements (lignes).

- Champs : nom et type de chaque colonne
- Valeurs : données contenues dans chaque colonne
- Clés : identifiants uniques pour chaque enregistrement

> [!TIP]
> Chaque table est comme un classeur Excel : les champs sont les colonnes, les lignes sont les enregistrements, et les clés primaires garantissent l’unicité.

> [!CAUTION]
> Ne jamais modifier les tables standard directement dans SE16N sans autorisation, car cela peut corrompre les données SAP.

## 🌺 BONNES PRATIQUES

- Toujours vérifier la table cible avant d’exécuter
- Limiter le nombre d’occurrences pour éviter de surcharger le système
- Utiliser les filtres pour analyser des données spécifiques
- Ne pas modifier directement les tables standard sans autorisation
- Exporter les données pour analyses ou reporting en dehors de SAP

> [!IMPORTANT]  
> SE16N est idéal pour la vérification, le contrôle et l’analyse des données SAP, ainsi que pour préparer des rapports simples pour Excel ou autres outils.

## 🌺 RESUME

> - SE16N permet d’accéder et de visualiser les données SAP sans programmation
> - Permet de filtrer, trier et rechercher des informations rapidement
> - Chaque table contient des champs (colonnes) et des enregistrements (lignes)
> - Idéal pour le contrôle des données et le reporting
> - Toujours suivre les bonnes pratiques pour éviter tout risque sur les données SAP
