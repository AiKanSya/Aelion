# 🌸 EXERCICES

## 🧩 EXERCICE 1 - AJOUTER DES MARGES

### 🍧 Énoncé

Énoncé :

Ajouter une marge responsive :

```xml
<Button text="Valider"/>
```

### 🍧 Correction :

```xml
<Button
    text="Valider"
    class="sapUiResponsiveMargin"/>
```
