# 🌸 I18N

## 🧩 I18N/ (INTERNATIONALISATION)

```
fgifirstappmodulename/
├── webapp/
│   ├── (annotations/)
│   ├── controller/
│   ├── css/
│   │
│   ├── i18n/ # <- Dossier d'Internationalisation
│   │
│   ├── libs/
│   ├── localService/
│   ├── model/
│   ├── view/
│   ├── Component.js
│   ├── index.html
│   └── manifest.json
│
├── .gitignore
├── (mta.yaml)
├── package-lock.json
├── package.json
├── README.md
├── ui5-local.yaml
├── ui5-mock.yaml
└── ui5.yaml
```

> [!IMPORTANT]
>
> - 🎯 Objectif
>
>   Gérer les textes de l’application dans plusieurs langues.
>
> - 🔨 Utilité : Séparer les textes affichés du code pour faciliter la traduction et la maintenance.
> - ⌚ Quand utilisé ? Dès qu’une application doit être utilisée par des utilisateurs de langues différentes.
> - 📌 Exemple :
>
>   Utiliser une clé de traduction dans une vue XML.

### 🍧 I18N.PROPERTIES (FICHIER PAR DÉFAUT) & I18N_EN.PROPERTIES (TRADUCTIONS ANGLAIS)

> [!IMPORTANT]
>
> - 🎯 Objectif
>
>   Définir les textes de l’application en anglais.
>
> - 🔨 Utilité : Afficher automatiquement l’interface en anglais
>   selon la langue de l’utilisateur.
> - ⌚ Quand utilisé ? Lorsque la langue de l’utilisateur est l’anglais.

📌 Exemple :

```properties
# This is the resource bundle for com.toray.omconfirmation

#Texts for manifest.json

#XTIT: Application name
appTitle                    =Confirmation by WH Order

#YDES: Application description
appDescription              =Confirmation by WH Order

#XTIT: Main view title
title                       =Confirmation by WH Order
title.Details               =Confirmation

#PLACE HOLDER
placeHolder.scanOM          =Scan WH Order

#ERROR
error.TextError             =No TM with this UM number is in progress
error.msgNoExistingHu       =HU not existing

#LIST
list.HeaderText             =WH Task
list.Vlenr                  =HU:
list.Who                    =WH Order:

#LABEL
label.Vlenr                 =Handling Unit
label.Matid                 =Product
label.Charg                 =Batch
label.Vlpla                 =Source Storage
label.Letyp                 =HU Type
label.Vsolm                 =Target Quantity
label.Meins                 =Base Unit of Measure
label.Nlpla                 =Destination Storage
label.Tanum                 =WH Task
label.Docno                 =Outbound Delivery Document Number
label.MovingVlenr           =Moving UM
label.LoadingVlenr          =Loading HU
label.ConfirmVlenr          =Confirmation HU
label.LowerVlenr            =Lower HU

#BUTTON
button.createPickHU         =Create Pal HU
button.cancel               =Cancel
button.confirm              =Confirm

#MESSAGETOAST
noDataFound                 =No data found.
dataRetrievingError         =Error retrieving data.
caseTypeError               =Untreated/anticipated case.
invalidData                 =Invalid data.
successfullConfirmation     =Task confirmed.
successfullPalletCreation   =The palette was successfully created.

#POPUP EMBALLAGE
embSel.title=Select one packaging
embSel.package=Packaging
embSel.ok=OK
embSel.cancel=Cancel
```

### 🍧 I18N_FR.PROPERTIES (TRADUCTIONS FRANÇAIS)

> [!IMPORTANT]
>
> - 🎯 Objectif
>
>   Définir les textes de l’application en français.
>
> - 🔨 Utilité : Fournir une version française de l’interface utilisateur.
> - ⌚ Quand utilisé ? Lorsque la langue de l’utilisateur est le français.

📌 Exemple :

```properties
# This is the resource bundle for com.toray.omconfirmation

#Texts for manifest.json

#XTIT: Application name
appTitle                    =Confirmation par Ordre Magasin

#YDES: Application description
appDescription              =Confirmation par Ordre Magasin

#XTIT: Main view title
title                       =Confirmation par Ordre Magasin
title.Details               =Confirmation

#PLACE HOLDER
placeHolder.scanOM          =Scan OM

#ERROR
error.TextError             =Aucune TM avec ce N° UM est en cours
error.msgNoExistingHu       =UM non existente

#LIST
list.HeaderText             =Tâche Magasin
list.Vlenr                  =UM :
list.Who                    =OM :

#LABEL
label.Vlenr                 =UM cédante
label.Matid                 =Article
label.Charg                 =Lot
label.Vlpla                 =Empl. cédant
label.Letyp                 =Type UM
label.Vsolm                 =Quantité
label.Meins                 =Unité
label.Nlpla                 =Empl. prenant
label.Tanum                 =Tâche magasin
label.Docno                 =N° Livraison sortante
label.MovingVlenr           =UM à déplacer
label.LoadingVlenr          =UM prenante
label.ConfirmVlenr          =Confirmation UM
label.LowerVlenr            =UM inférieure

#BUTTON
button.createPickHU         =Créer UM Pal
button.cancel               =Annuler
button.confirm              =Confirmer

#MESSAGETOAST
noDataFound                 =Aucune donnée trouvée.
dataRetrievingError         =Erreur lors de la récupération des données.
caseTypeError               =Cas non traité/anticipé.
invalidData                 =Données invalides.
successfullConfirmation     =Tâche confirmée.
successfullPalletCreation   =La palette a été créé avec succés.

#POPUP EMBALLAGE
embSel.title=Selectionner un emballage
embSel.package=Emballage
embSel.ok=OK
embSel.cancel=Annuler
```

📌 Exemple :

```xml
<Title text="{i18n>appTitle}"/>
<Button text="{i18n>button.confirm}"/>
<Input placeholder="{i18n>placeHolder.scanOM}"/>
```
