## 🌺 INSTALLATION MANUELLE

Référence : https://draculatheme.com/abap

0. 🍧 Fermer complètement SAP GUI

1. 🍧 Télécharger le zip : https://github.com/dracula/abap/archive/master.zip

2. 🍧 Dézipper le dossier

3. 🍧 Dans le dossier/abap-master, récupérer le fichier abap_spec.xml

4. 🍧 Appuyer sur [ Win ] + [ R ] et renseigner `%APPDATA%` dans le champ

   ![](./assets/images/Capture%20d’écran%202025-11-04%20130735.png)

5. 🍧 Dans le dossier `SAP`/`SAP GUI`/`ABAP Editor`...

   ![](./assets/images/Capture%20d’écran%202025-11-04%20130904.png)

6. 🍧 Créer une copie du fichier original `abap_spec.xml` (backup au cas où)

7. 🍧 Remplacer le fichier `abap_spec.xml` par celui du dossier zip

8. 🍧 Lancer SAP GUI pour visualiser le changement de thème
