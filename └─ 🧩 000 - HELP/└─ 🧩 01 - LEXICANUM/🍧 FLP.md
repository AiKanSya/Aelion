# 🌸 SAP FIORI LAUNCHPAD (FLP)

## 🧩 DEFINITION

Le `Fiori Launchpad (FLP)` est le portail central qui regroupe toutes les applications Fiori accessibles à un utilisateur.  
Il fournit un point d’entrée unique pour naviguer entre les différentes applications, organiser les tuiles, gérer les notifications et accéder aux rôles et autorisations.

> [!TIP]
> Imagine le FLP comme le tableau de bord de ton smartphone, avec toutes tes applications rangées sous forme de tuiles et prêtes à être ouvertes en un clic.

## 🧩 UTILITE

Le FLP simplifie la vie des utilisateurs : il centralise toutes les applications, permet de personnaliser l’affichage, de rechercher rapidement des applications et de recevoir des alertes importantes.

> [!TIP]
> Concrètement, c’est l’écran d’accueil de SAP : tu vois toutes les applications que tu peux utiliser et tu navigues facilement entre elles.

## 🧩 POINTS IMPORTANTS

- Regroupe toutes les applications Fiori accessibles selon les rôles de l’utilisateur.
- Affiche les applications sous forme de tuiles (tiles) personnalisables.
- Permet la recherche, les notifications et la navigation rapide.
- Point d’entrée unique pour toutes les applications SAP Fiori.

> [!TIP]
> FLP = ton tableau de bord central pour lancer et naviguer dans toutes tes applications Fiori.
