# 🌸 BASECONTROLLER

## 🧩 DEFINITION

`BaseController` est une classe parent pour tous les controllers d’une application, contenant des fonctions communes réutilisables.

> [!TIP]
> Imagine BaseController comme la boîte à outils partagée pour tous tes chefs d’orchestre, avec des fonctions de navigation, d’affichage ou de gestion des messages.

## 🧩 UTILITE

Elle évite de répéter le même code dans chaque Controller et facilite la maintenance et l’évolution de l’application.

> [!TIP]
> Concrètement, BaseController fournit les fonctions que tu peux appeler depuis n’importe quel Controller pour simplifier ton code.

## 🧩 POINTS IMPORTANTS

- Hérité par tous les Controllers de l’application.
- Contient fonctions de navigation, utilitaires et helpers.
- Standardise le comportement des Controllers.
- Permet une maintenance plus simple.

> [!TIP]
> BaseController = les fonctions partagées pour tous tes écrans.
