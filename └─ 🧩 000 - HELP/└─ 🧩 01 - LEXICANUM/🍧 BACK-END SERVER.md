# 🌸 BACK-END SERVER (BES)

## 🧩 DEFINITION

Le `Back-End Server` (BES) est le serveur qui contient toutes les données et la logique SAP.  
Il gère les informations, effectue les calculs et traite les requêtes envoyées par le `Front-End Server (FES)`.

> [!TIP]
> Imagine le `BES` comme la cuisine d’un restaurant : le `FES` prend la commande du client, mais la cuisine prépare le plat.

## 🧩 UTILITE

Le `BES` permet de stocker, organiser et traiter les données SAP de façon sécurisée et fiable.  
Il supporte toutes les fonctions complexes derrière les applications `Fiori/UI5`.

> [!TIP]
> Comme une banque : elle garde l’argent (données) en sécurité et effectue les opérations demandées par les clients (FES).

## 🧩 POINTS IMPORTANTS

- Contient toutes les tables, vues et programmes ABAP nécessaires au fonctionnement de SAP.
- Traite les requêtes reçues du Front-End Server et renvoie les réponses.
- Assure la sécurité et l’intégrité des données SAP.

> [!TIP]
> Le `Back-End Server` = la "cuisine" et le "coffre-fort" de SAP : c’est là que tout se prépare avant d’être servi à l’utilisateur.
