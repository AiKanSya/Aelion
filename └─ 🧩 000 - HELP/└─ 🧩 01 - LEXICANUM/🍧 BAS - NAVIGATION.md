# 🌸 NAVIGATION

## 🧩 DEFINITION

La `Navigation` est la logique qui permet de passer d’une View à une autre dans une application Fiori.

> [!TIP]
> Imagine la navigation comme les couloirs d’un immeuble, qui te permettent d’aller d’une pièce à l’autre.

## 🧩 UTILITE

Elle organise le parcours utilisateur et permet de créer des applications multi-écrans fonctionnelles et cohérentes.

> [!TIP]
> Concrètement, c’est ce qui fait que cliquer sur un bouton t’emmène vers la bonne page de l’application.

## 🧩 POINTS IMPORTANTS

- Utilisée dans les Controllers.
- Définie dans le `manifest.json` via les routes.
- Permet une expérience utilisateur fluide.
- Partie essentielle du design Fiori multi-écran.

> [!TIP]
> Navigation = déplacer l’utilisateur dans ton application.
