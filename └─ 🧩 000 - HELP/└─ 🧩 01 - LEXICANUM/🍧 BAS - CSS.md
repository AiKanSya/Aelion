# 🌸 CSS

## 🧩 DEFINITION

`CSS` est le langage qui définit le style et l’apparence des éléments dans une application UI5/Fiori.

> [!TIP]
> Imagine CSS comme les peintures et décorations dans ta maison, qui rendent tout joli et lisible.

## 🧩 UTILITE

CSS permet de personnaliser l’apparence des boutons, tableaux, textes et autres composants pour rendre l’interface agréable et cohérente.

> [!TIP]
> Concrètement, CSS change la couleur, la taille, la police ou la disposition des éléments de ton application.

## 🧩 POINTS IMPORTANTS

- Travaille avec les Views XML et UI5.
- Permet de respecter la charte graphique SAP.
- Compatible avec responsive design.
- Ne touche pas à la logique ou aux données.

> [!TIP]
> CSS = l’apparence de ton application.
