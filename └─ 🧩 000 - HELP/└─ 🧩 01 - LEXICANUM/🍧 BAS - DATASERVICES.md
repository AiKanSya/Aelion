# 🌸 DATASERVICES

## 🧩 DEFINITION

`DataServices.js` sont **les services utilisés pour communiquer entre l’application Fiori et le backend SAP**, souvent via OData.  
Ils permettent de récupérer, créer, modifier ou supprimer des données.

> [!TIP]
> Imagine DataServices comme **le livreur entre le réfrigérateur (backend) et la table (View)**, qui apporte les ingrédients et renvoie les plats préparés.

## 🧩 UTILITE

Ils fournissent les données nécessaires aux Models et assurent que les informations envoyées par l’utilisateur sont correctement traitées par le backend.

> [!TIP]
> Concrètement, DataServices font le lien entre ton application et les données SAP.

## 🧩 POINTS IMPORTANTS

- Utilisés par les Models pour charger ou envoyer des données.
- Fonctionnent généralement via OData.
- Indispensables pour toutes les applications Fiori qui manipulent des données.
- Centralisent la communication avec le backend.

> [!TIP]
> DataServices = le lien entre ton application et SAP pour toutes les données.
