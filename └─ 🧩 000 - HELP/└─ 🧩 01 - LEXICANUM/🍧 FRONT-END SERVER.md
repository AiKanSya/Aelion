# 🌸 FRONT-END SERVER (FES)

## 🧩 DEFINITION

Le `Front-End Server` (FES) est le serveur qui héberge les applications Fiori/UI5 et gère l’interface utilisateur.  
Il reçoit les demandes des utilisateurs, les envoie au système SAP (Backend), et renvoie les réponses à l’application.

> [!TIP]
> Imagine le `FES` comme la réception d’un hôtel : il accueille les visiteurs (utilisateurs), transmet leurs demandes aux bonnes personnes (Backend), et leur rapporte les réponses.

## 🧩 UTILITE

Le `FES` permet aux utilisateurs de voir et interagir avec les applications SAP via un navigateur ou un appareil mobile, sans avoir à accéder directement au système SAP complexe.

> [!TIP]
> Comme une interface de contact : tu parles au réceptionniste (FES), il fait le travail compliqué en coulisses et te rapporte le résultat.

## 🧩 POINTS IMPORTANTS

- Le `FES` contient le SAP Gateway, les applications Fiori/UI5 et les bibliothèques nécessaires.
- Il communique avec le Backend SAP pour récupérer ou modifier les données.
- Permet de gérer la sécurité et les droits d’accès des utilisateurs.

> [!TIP]
> Le `Front-End Server` = le guichet unique pour l’utilisateur : tout passe par lui pour que les interactions soient simples et sécurisées.
