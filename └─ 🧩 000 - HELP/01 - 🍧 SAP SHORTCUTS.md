# 🌸 SAP SHORTCUTS

## 🌺 DEVELOPMENT

| 🍧 RACCOURCIS                   | 🍧 EXPLICATION                    |
| ------------------------------- | --------------------------------- |
| ALT (dans un programme)         | Sélection verticale               |
| CTRL + Y (dans une transaction) | Sélection                         |
| CTRL + D (dans un programme)    | Duplique la ligne sélectionnée    |
| CTRL + F                        | Rechercher                        |
| CTRL + G                        | Rechercher la prochaine itération |
| CTRL + C                        | Copier                            |
| CTRL + X                        | Couper                            |
| CTRL + V                        | Coller                            |
